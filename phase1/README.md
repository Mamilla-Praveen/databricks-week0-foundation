# Phase 1 - Basics of SQL and PySpark

**Objective**
The aim of this phase is to learn the basic concepts of SQL and get an introduction to PySpark. This phase focuses on understanding how to work with data in a simple and clear way.

**Exercises Completed**
In this phase, I worked on basic SQL queries such as selecting data, filtering rows using conditions, and sorting results. I also practiced creating and working with simple DataFrames in PySpark and performed basic operations on them.

**Key Learnings**
I learned how to write SQL queries to get data from a table and apply conditions to it. I understood how sorting works and how to select only the required columns. In PySpark, I learned how DataFrames work and how to perform simple operations like selecting, filtering, and displaying data.

**Challenges Faced**
At the beginning, it was a bit confusing to understand the SQL syntax. I also faced some difficulty in understanding how PySpark works compared to SQL. Fixing errors in the code and understanding the output took some time.
