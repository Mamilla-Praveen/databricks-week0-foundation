# Week 0 – SQL, Databricks & PySpark Introduction

## Objective
The objective of this week is to build a strong foundation in data engineering by understanding core concepts of SQL, exploring the Databricks platform, and getting introduced to PySpark. This phase focuses on learning how data is stored, queried, and processed in both traditional and big data environments. It also emphasizes hands-on practice to bridge the gap between theory and real-world implementation.

---

## Learning Approach
This week was focused on a structured learning approach where I combined theoretical understanding with practical implementation. I followed a step-by-step process:
- First, I understood the concepts
- Then, I practiced them using small datasets
- Finally, I executed them in the Databricks environment

This helped me build confidence and clarity in each topic.

---

## Topics Covered

### SQL Introduction
- Learned the importance of SQL in data analysis and data engineering
- Understood relational databases, tables, rows, and columns
- Practiced core SQL operations:
  - SELECT for retrieving data
  - WHERE for filtering conditions
  - ORDER BY for sorting results
  - LIMIT for restricting output size
- Explored logical operators like AND, OR, and NOT
- Understood NULL values and basic handling techniques
- Practiced writing queries on structured datasets
- Learned how to read and interpret query outputs

---

### Databricks Platform Introduction
- Understood Databricks as a unified data analytics platform
- Explored workspace features such as:
  - Notebooks
  - Clusters
  - Data storage
- Learned how to:
  - Create and organize notebooks
  - Attach clusters and run computations
  - Use different languages within the same notebook
- Executed commands using:
  - %sql for SQL queries
  - %python for PySpark code
- Understood how Databricks simplifies big data processing
- Worked with sample datasets available in the platform
- Learned basic workflow management inside Databricks

---

### PySpark Introduction
- Understood what PySpark is and why it is used for large-scale data processing
- Learned about Spark architecture at a basic level
- Initialized SparkSession and understood its role
- Loaded data from CSV files into DataFrames
- Explored DataFrame operations:
  - show() to display data
  - printSchema() to understand structure
  - select() to choose columns
  - filter() to apply conditions
- Understood immutability of DataFrames
- Learned the difference between transformations and actions
- Compared SQL queries with PySpark operations
